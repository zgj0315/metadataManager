# 照片管理工具
## 简介
处理照片文件

## 需求
1. 按照拍摄时间，分类存储
2. 去除重复的照片
3. 文件修改时间改为拍摄时间

## 技术点
- [x] 读取图片文件的创建时间 
- [x] 计算文件md5
- [x] 计算文件大小

## 方案设计
### 术语约定
Original：源文件，相机输出的cr2或jpg  
Export：输出文件，LR软件处理后的图片
